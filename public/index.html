API Keys Required — Add to Vercel Environment Variables
Go to your Vercel project → Settings → Environment Variables and add:
PROPERTYDATA_API_KEY — from propertydata.co.uk/free-trial
PLANNING_API_KEY — from api.planning.org.uk/v1/generatekey (free, instant)
COMPANIES_HOUSE_API_KEY — from developer.company-information.service.gov.uk
BREVO_API_KEY — from app.brevo.com (for digest emails)
DIGEST_EMAIL — your email address for morning digests
